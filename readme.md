# LOP DAO Event Reader

This is an event listener to get all transactions event of LOP smart contract transactions.
All transactions are saved in AWS `DynamoDB` with AWS `Lambada`

## Environment

- Node version (^ 16.0.0)
- Yarn version (^ 1.22.0)

## Running Project

### Install dependency

`yarn install`

### Run

`yarn start`
